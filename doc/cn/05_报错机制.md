### 路径载入错误

目录结构:

* `index.html`
* `js`
	* `sr.js`
	* `main.js`

code:

```javascript
sr.config({
    baseUrl: "js/"
});

//main.js
require('aa')
	.done(function(data){
		console.log(data);
	})
	.fail(function(e){
		console.log(e);
	});
```

打印结果：

```
Array[1]
	0: Object
	  no: 0
	  state: "error"
	  type: "error"
	  url: "js/aa.js"
	  __proto__: Object
	length: 1
	__proto__: Array[0]
```

说明：

因为 `aa` 是不存在的，在获取失败的情况下，会执行fail内的函数。
